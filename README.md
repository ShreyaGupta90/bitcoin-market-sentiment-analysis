# 📈 Bitcoin Market Sentiment Analysis

Analyze the relationship between Bitcoin market sentiment and trader performance using historical trading data and the Fear & Greed Index.

---

## 📖 Project Overview

Market sentiment often influences trading decisions. This project combines historical cryptocurrency trading data with the Bitcoin Fear & Greed Index to explore how different market conditions relate to trader behaviour and profitability.

The project focuses on data cleaning, preprocessing, dataset merging, exploratory data analysis (EDA), and deriving meaningful insights through visualizations.

---

## 🎯 Objectives

- Clean and preprocess trading and sentiment datasets
- Merge datasets using trading dates
- Analyze trader performance under different market sentiments
- Visualize trading behaviour using charts
- Generate insights that can support data-driven trading decisions

---
