# Bitcoin Market Sentiment Analysis

## Overview

This project analyzes Bitcoin market sentiment by combining trading data with the Fear & Greed Index. The objective is to understand how different market sentiments relate to trader performance and trading activity.

## Dataset

The analysis uses two datasets:

- Historical Trading Data
- Bitcoin Fear & Greed Index

## Technologies Used

- Python
- Pandas
- Matplotlib
- Google Colab

## Project Workflow

- Load datasets
- Clean and preprocess data
- Convert date columns
- Merge trading and sentiment datasets
- Perform exploratory data analysis
- Visualize results using charts
- Summarize key observations

## Key Insights

- Trading performance varies across different market sentiments.
- Trade distribution changes during Fear, Greed, and Neutral market conditions.
- Market sentiment provides useful context for understanding trading behaviour.
